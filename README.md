# 👥 Gestión de Personas - Java Swing

Trabajo Práctico de **Laboratorio IV** — Universidad Tecnológica Nacional.
Aplicación de escritorio para la gestión de personas de una empresa, desarrollada en **Java con Swing** y arquitectura de **tres capas con interfaces**.

## 🛠️ Tecnologías

![Java](https://img.shields.io/badge/java-%23ED8B00.svg?style=for-the-badge&logo=openjdk&logoColor=white)
![MySQL](https://img.shields.io/badge/mysql-%2300f.svg?style=for-the-badge&logo=mysql&logoColor=white)

## 📌 Funcionalidades

- Agregar, modificar, eliminar y listar personas
- Validaciones de campos en tiempo real con KeyListener
- Listado en JTable con datos desde la base de datos
- Conexión y persistencia en MySQL

## 🏗️ Arquitectura

Proyecto organizado en tres capas con interfaces:
- **Presentación** — interfaz gráfica con Java Swing
- **Negocio** — lógica de la aplicación
- **DAO** — acceso a datos con MySQL
